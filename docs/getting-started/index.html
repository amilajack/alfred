Getting Started

#Usage

# Install alfred
npm install alfred --global --registry https://amilajack.com/registry


# Create a new project
alfred new my-project
cd my-project


# Build your project
alfred run build

For migrating to Alfred, see the migrating guide

#Learning Skills

# Learning skills
alfred learn @alfred/skill-react @alfred/skill-redux
