## Installing

```bash
# NPM
npm install --global @alfredpkg/cli
# Yarn
yarn global add @alfredpkg/cli

alfred new my-new-project
cd my-new-project
```

## Running Subcommands

```bash
alfred start
alfred build
alfred clean
alfred format
alfred test
alfred search
alfred doc
alfred migrate
alfred types
alfred migrate
```

## Learning New Skills

```bash
# Learning skills
alfred learn alfred-skill-build-parcel
# Build using the new subcommand
alfred build
# Learning multiple skills
alfred learn alfred-skill-build-angular alfred-skill-build-redux
```
